🚀 주요 기능

  • 실시간 데이터 수집: NCM Al Bahar 페이지 Selenium 크롤링
  • 오프라인 모드 ⭐: API 키 누락 시 자동 합성 데이터 생성
  • Resilience 메커니즘 ⭐: 각 데이터 소스별 독립적 fallback 처리
  • 벡터 기반 검색: "AGI high tide RORO window" 자연어 질의
  • 운항 판정: GO/CONDITIONAL/NO-GO 자동 분류
  • 자동화: 3시간마다 수집 + 알림 시스템
  • 통합 분석: 다중 소스 예보 융합 및 ERI 계산
  • 실행 모드 선택 ⭐: auto/online/offline 모드 지원

🎯 비즈니스 가치

  • 운영 효율성: 자동화된 데이터 수집 및 분석
  • 시스템 안정성 ⭐: API 키 없이도 정상 작동 (오프라인 모드)
  • 개발자 경험 ⭐: 즉시 테스트 가능 (API 키 불필요)
  • 안전성 향상: 실시간 해양 조건 모니터링
  • 비용 절감: 연료 효율 최적화 및 지연 감소
  • 의사결정 지원: AI 기반 운항 조건 분석
  • 규정 준수: FANR/MOIAT 규정 자동 검증
  • CI/CD 친화적 ⭐: GitHub Actions 안정적 동작 보장
